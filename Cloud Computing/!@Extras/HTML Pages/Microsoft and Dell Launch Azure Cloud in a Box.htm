Windows Server Migration

Microsoft and Dell Launch Azure Cloud in a Box

Howard M. Cohen | Date: 2014-10-22

 0  4  11  0
  • Print PDF
  • Main Category: Windows Server Migration

With pre-integrated hardware from Dell and software from Microsoft, the Microsoft Cloud Platform System delivers an on-premises appliance that offers a self-service portal for managing services like Web sites, virtual machines, and Service Bus; a portal for administrators to manage resource clouds; scalable Web hosting; and more.


    For those of you who still think that “cloud” means that the services come from somewhere other than your own data center, this would be a good time to re-read the Definition of Cloud Computing from the National Institute of Standards & Technology (NIST) which define them based on specific characteristics like elasticity, user self-service, metered service and more, but say nothing about locale of the hardware or software.

    This is important right now because Microsoft and Dell just unveiled “The Microsoft Cloud Platform System.”

    Cloud in a Box

    The Microsoft October 20, 2014 press release announcing the new introduction explains that “the Cloud Platform System (CPS), powered by Dell, brings together Azure, Windows Server and Microsoft System Center to deliver an ‘Azure-consistent cloud in a box.’ With pre-integrated hardware from Dell and software from Microsoft, CPS delivers learnings from Azure to customers and partners with the control of an on-premises appliance. CPS will be available for purchase on Nov. 3.”

    Originally a project code named “San Diego”, ZDNet Microsoft follower Mary Jo Foley reports that, “With Dell as its hardware partner, Microsoft will offer customers pre-assembled racks of servers running Windows Server 2012 R2, System Center 2012 R2 and Windows Azure Pack. Azure Pack, originally known as ‘Windows Azure Services for Windows Server,’ provides users with the on-premises equivalents of a number of Azure technologies, including a self-service portal for managing services like Web sites, virtual machines and Service Bus; a portal for administrators to manage ‘resource clouds’; scalable Web hosting and more.”

    Private vs. Public Cloud – The Gap Widens and Becomes More Obvious

    For many users who may still be confused about the difference between public and private cloud, this announcement should further emphasize the dramatic difference between the two.  While Public Cloud Services are indeed supplied by a remote data center, Private Cloud describes servers with specific characteristics that make it far more convenient and facile, including user self-service portals, rapidly requested and released resources, virtual machine and other management enhancements.  All of these are featured in the Microsoft Cloud Platform System.
