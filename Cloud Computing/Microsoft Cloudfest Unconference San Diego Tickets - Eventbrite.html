      Event Details

      Microsoft Cloudfest Unconference - San Diego
      Join us on Wednesday November 5th at the Microsoft Office in San Diego for a Cloudfest Unconference! If you've never been to an Unconference-style event before... Unconferences are driven by you, the participants. We'll bring a few speakers that are extremely knowledgable around the topic areas of discussion and they will be prepared to demonstrate & train participants on the topics below, but we're also opening up the speaking opportunities to you all, the subject matter experts within the industry that are using these technologies day-in and day-out.


      Topic Areas for Discussion
      1. Azure Websites
      How to create them, how to debug them, how to move them around various slots (i.e. development, staging, production), etc. How to manage data, and how to manage data when working with slots.

      2. Mobile Backend Services within Azure
      How to create them, how to manage identity, how to use the .NET APIs for Azure Active Directory.

      3. DevOps & Automation
      How to automate with Azure Powershell & DSC.

      4. Working with Cloud Services
      How do they differ from just traditional VMs? When to use Cloud Services over Azure Websites?
       
      5. Virtual Machines 
      How to configure them, how to start them via Visual Studio, best practices for management.

      6. Security Best Practices 
      Tooling to secure and store transactional information to and from apps.

      Have suggestions/requests for more? Do you have a topic that you would like to speak on? Contact me and lets talk about bringing you on as a speaker for this event. My contact details can be found at www.microsoftappdev.com.

      9:00am - Start Time
