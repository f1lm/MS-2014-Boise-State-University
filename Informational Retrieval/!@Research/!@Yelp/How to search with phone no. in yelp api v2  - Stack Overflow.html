    On the 1.0 API (http://www.yelp.com/developers/documentation/phone_api) you are able to do it.
    I'd suspect you can on the 2.0 too I'd try it out.
    here is what they say:
    http://api.yelp.com/phone_search?phone=1234567890&ywsid=XXXXXXXXXXXXXXXX
    I'd try on api 2.0
    http://api.yelp.com/v2/search?phone_search=XXXXXXXXXX
    or
    http://api.yelp.com/v2/search?phone=XXXXXXXXXX
    And see if it works.
    To make the answer short, YES you can but only for sure on the 1.0 API and MAYBE on the 2.0
