Markup user interface components like HTML creating components with database connection. Numerious data components ({@link de.must.markup.Storable}) are available to be laid-out in special frame-like container for querying and administrating database tables. A similar developing toolkit for Swing UI is available with package {@link de.must.wuic}

Concept: Think in java objects and let the framework generate what you need, e.g. HTML. If Java Server Pages are wished, you may also benefit from the package markup by using e.g. {@link de.must.markup.GroupOfStorables}
