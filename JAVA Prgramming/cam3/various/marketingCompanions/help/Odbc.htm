Adressverwaltung

 Anleitung zur Einrichtung
der Adress-Datenbank

Vorbemerkung

Produkte von MÜLLER UND STEIN software der neuen Generation sind in Java entwickelt. Dies erleichtert die freie Wahl der Datenbank. In der Standardversion erfolgt der Zugriff über ODBC auf eine Access-Datenbank. Diese muss nach dem Setup der Inventarverwaltung eingerichtet werden.
 

1. Aufruf der ODBC-Administration über Systemsteuerung

Rufen Sie innerhalb der Systemsteuerung die ODBC-Konfiguration mit folgendem Symbol auf:

2. Wählen Sie System DSN drücken Sie Button 'Add' bzw. 'Hinzufügen'


 
 

3. Den gewünschten Datenbanktreiber wählen, in diesem Beispiel ist es der Microsoft Access-Treiber.



 

Button 'Fertig stellen' drücken.

4. Angaben zur Datenbank machen

  • Data Source Name 'Marketing'
  • Button 'Create' bzw. 'Erstellen' drücken
  • Verzeichnis wählen, z.B. das dafür vorgesehen Unterverzeichnis 'databases' der Adressverwaltung oder ein Verzeichnis auf dem gewünschten Server.
  • Datenbankname angeben, z.B. Marketing.mdb
  • OK drücken (alle verbleibenden Formulare)
Alle in dieser Weise konfigurierten ODBC-Verbindungen können jetzt in der Adressverwaltung benutzt werden (bei Bedarf unterschiedliche Mandanten). Die Auswahl erfolgt über Menü Administration, Basis-Parameter, Datenbank, Datenquellenname.

Hinweis: Für die gemeinsame Nutzung einer Datenbank auf einem Server ab der zweiten Arbeitsplatzinstallation statt 'Datenbank erzeugen' 'Datenbank auswählen' verwenden.
