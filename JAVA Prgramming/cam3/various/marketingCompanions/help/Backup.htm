Datensicherung

Bitte führen Sie regelmäßige Datensicherungen durch. Im einfachsten, speziellen Fall bedeutet dies, die Datei C:\Programme\Marketing\databases\Marketing.mdb auf Diskette zu kopieren. Für die organisierte Sicherung, z.B. automatisierte Server-Sicherung, folgende Hinweise:
  • Alle verwendeten ODBC-Verbindungen der Adressverwaltung haben eine eigene Datenbank-Datei (.mdb)
  • Im ODBC-Datenquellen-Administrator wurde seitens des Anwenders definiert, welche Datenbank damit bearbeitet wird.
  • Alle Adressdaten werden in diesen Datenbanken gespeichert. Deshalb kann sich die regelmäßige Sicherung auf diese Dateien beschränken. Basisparameter werden in "\bin\ParmV001.ser" und individuelle enstergrößen in "\bin\<NT-user-name>.wpr" gespeichert.
