Kochbuch

Einführung

Das Kochbuch ist eine exemplarische Anwendung. Es demonstriert Server-Anwendungen mit Verwaltungsfunktionen. Diese umfassen insbesondere das Anlegen, Suchen und Ändern von Einträgen in verschiedenen Tabellen. Dabei stehen unterschiedliche Funktionen und Berechtigungen zur Verfügung, je nachdem ob und als welcher Benutzer man sich angemeldet hat.

Im Falle eines Produkts ist an dieser Stelle eine fachliche Einführung in die jeweilige Anwendung vorgesehen. Im folgenden sei das in knapper Form angedeutet:

Das Kochbuch ist eine Plattform zum internationalen Austausch von Kochrezepten.

Weitere Möglichkeiten, Hilfe zu erhalten

Kurzer Hinweis je nach Mauszeiger-Position Eventuelle kurze Hilfen zu einem Feld werden in der Windows-Statuszeile (ganz unten) angezeigt, wenn sich der Cursor über diesem Feld befindet.
Kontext-Hilfe über Button "Hilfe" Wenn ein Button "Hilfe" angezeigt wird, dann steht eine spezielle Hilfe für diesen Kontext zur Verfügung und kann mit Click auf diesen Button angefordert werden.
